SOBRE O PROJETO GREEN DAY

TITLE

O projeto Green Day é um robô que regar automaticamente um certo tipo de planta e mandar um relatório para o usuário, esse relatório consiste em informar a umidade ideal para a planta e como também qual solo é adequada para ela, mas qual seria o seu diferencial em relação a outros projetos? A capacidade de se autorregular.

O projeto Green Day tem uma câmera acoplada a ele que permite tira fotos, através de um cálculo chamado Índice de Similaridade Estrutural (ISE) o Arduino vai fazer uma comparação entre a ultimo frame com um banco de dados MySQL, se o resultado do cálculo for superior a um certo número, ele vai se autorregular e fazer os seus afazeres.
